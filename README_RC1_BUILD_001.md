# GT7 Championship Manager — RC1 Build 001

Primera build funcional orientada a validar el flujo principal:

1. Crear Garaje GT7.
2. Crear campeonato.
3. Revisar/editar plantilla GT7.
4. Abrir Race Control.
5. Registrar resultado.
6. Ver clasificación actualizada.

## Instalación en GitHub Pages

Sube al repositorio el contenido de la carpeta `gt7_rc1_build001`, no el ZIP.

Archivos esperados en la raíz del repositorio:

- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `css/`
- `js/`
- `data/`
- `icons/`
- `docs/`

Después abre:

`https://jdm1822.github.io/GT7-Championship-Manager/?build=001`

En iPhone, desde Safari: Compartir → Añadir a pantalla de inicio.

## Nota importante

La base de datos de coches incluida en esta Build es amplia para pruebas, pero todavía no es definitiva ni completa al 100 %. La RC1 Final ampliará y depurará el catálogo.
