# GT7 Championship Manager — RC1 Build 001

Primera build funcional real de la RC1.

## Instalación
Sube todos estos archivos a la raíz del repositorio GitHub Pages y abre:

`https://jdm1822.github.io/GT7-Championship-Manager/?build=rc1-001`

## Incluye
- Dashboard orientado a próxima carrera.
- Garaje GT7 permanente.
- Base inicial de coches ampliable.
- Constructor de campeonatos.
- Plantilla GT7.
- Race Control.
- Registro de resultados.
- Clasificación automática.
- Exportar/importar datos.
- PWA instalable.

Designed by JDM & ChatGPT.
