# GT7 Championship Manager · RC1 Build 002

Build centrada en ampliar la base de circuitos y coches.

## Cómo instalar

1. Borra los archivos actuales del repositorio.
2. Sube el contenido de la carpeta `gt7_rc1_build002`.
3. Abre la web con `?build=rc1-002`.
4. Si la PWA antigua queda cacheada, borra datos de `github.io` en Safari.

## Nota

La base de coches queda ampliada, pero sigue siendo evolutiva. En siguientes builds iremos corrigiendo ausencias concretas que detectes al crear tu Garaje GT7.
