# Changelog

## RC1 Build 003

### Añadido
- Separación entre Base de datos GT7 y Garaje GT7.
- PR base y potencia en fichas de coche.
- Selector de coche desde el Garaje GT7 durante la creación de campeonato.
- Calendario editable carrera a carrera con variantes/layouts.
- Vista completa del campeonato con pilotos, coches, reglas y calendario.
- Race Control clarificado: “Registrar resultado de: Carrera X · Circuito · Variante”.

### Mejorado
- Flujo de creación de campeonato.
- Integración lógica con plantillas de carrera personalizada de GT7.
- Documentación simplificada para mantener una única versión actual.

### Limitaciones conocidas
- La base de datos de coches sigue siendo ampliable y no debe considerarse todavía 100% completa.
- El registro de resultados calcula de momento el resultado del usuario; la clasificación completa de 20 pilotos se ampliará en la siguiente iteración.
